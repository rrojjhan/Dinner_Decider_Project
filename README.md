# 🍽️ Dinner Decider Project

> A beginner-friendly Python project that helps you decide what to eat for dinner when you just can't choose.

## 📖 Project Overview

The **Dinner Decider** is a fun Python program that asks the user a few simple questions about what they are craving, then suggests a delicious meal based on their answers.

This project was created as part of an **Introduction to Programming** course at **Barnes Academy High School**.

👩‍🏫 **Mentor:** Miss Rachnicha
🐍 **Programming Language:** Python
📚 **Concepts Used:** Functions, Loops, Conditionals, Lists, Dictionaries, JSON Files, and Random Selection

---

## ✨ Features

* 🥩 Choose a food category (Meat, Veggie, Both, or Dairy)
* 🍗 Select a protein
* 🥦 Pick vegetables
* 🍚 Choose a starch
* 🧂 Add toppings if desired
* 🎲 Receive a random dinner suggestion
* 🔄 Ask for another suggestion

---

## 🧠 Programming Concepts Practiced

This project is great for beginners because it uses many important Python topics:

| Concept                | How It Is Used                        |
| ---------------------- | ------------------------------------- |
| `input()`              | Gets answers from the user            |
| `print()`              | Displays messages                     |
| `if/elif/else`         | Makes decisions                       |
| `while` loops          | Repeats until valid input is entered  |
| Functions              | Organizes the code into smaller parts |
| Lists and Dictionaries | Stores meal information               |
| JSON                   | Saves and loads meal data             |
| `random.choice()`      | Picks a random meal                   |

## ▶️ How to Run the Program

1. Install Python (version 3.8 or higher).
2. Download the project files.
3. Open a terminal or command prompt.
4. Navigate to the project folder.
5. Run the program:

```bash
python Dinner_Decider_Project.py
```

---

## 🔄 Program Flow

1. Load meal data from a JSON file.
2. Ask the user questions.
3. Filter meals that match the user's choices.
4. Randomly select one matching meal.
5. Display the dinner suggestion.
6. Ask whether the user wants another suggestion.
