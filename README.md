# Dinner_Decider_Project
Project Overview: 
The Dinner Decision Maker is a Python program that helps users choose what to eat when they can’t decide. 
The app suggests a random meal from a list of options or categories the user provides. 
It’s designed for anyone who wants quick, stress‑free dinner ideas
